# Unit Tests and Integration Tests

This project contains unit tests and integration tests for a GitHub organization client utility module. It demonstrates testing practices using Python's `unittest` framework and the `parameterized` library for parameterized test cases.

## Files

- `utils.py` - Generic utilities for GitHub org client including nested map access, JSON retrieval, and memoization decorator
- `client.py` - GitHub organization client implementation
- `test_utils.py` - Unit tests for utility functions
- `fixtures.py` - Test fixtures and mock data
